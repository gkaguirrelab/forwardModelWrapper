-retinotopy_results.mat = Retinotopy results .mat file

-angle_map.dtseries.nii = Angle maps in 360-360 scale

-eccentricity_map.dtseries.nii = Eccentricty maps 

-exponent_map.dtseries.nii = Exponent maps

-gain_map.dtseries.nii = Gain maps

-R2_map.dtseries.nii = Weight maps

-rfsize_map.dtseries.nii = pRF size

-x_map.dtseries.nii = Maps are converted to Cartesian coordinates from Polar coordinates before the FSLR to native interpolation. This is the map for x-ccordinates

-y_map.dtseries.nii = Maps are converted to Cartesian coordinates from Polar coordinates before the FSLR to native interpolation. This is the map for y-ccordinates


** If you set a threshold on the gear configuration page, another set of maps will be created for each of these files above.
